# Phone Bot - Tự động đọc số điện thoại & gửi tin Zalo

## Tính năng
- ✅ Bot Telegram: Nhận ảnh, OCR đọc số điện thoại
- ✅ Lưu số vào Google Sheets
- ✅ Bot Zalo: Tự động gửi lời mời kết bạn + tin nhắn sau 30 phút

## Cấu trúc
## Cách sử dụng
1. Sửa file `.env` với thông tin của bạn
2. Push lên GitHub
3. Deploy lên Railway
4. Gửi ảnh vào Telegram Group
